### better5e

This is a project I will be working on over the next year or so. Goal is to create a desktop app campaign manager for 5e games. 
This will be a completely customizable experience, allowing users to create classes (suck it WOTC), subclasses, features and items, and then build their characters with them. We then export a character sheet in HTML to roll with Beyond20 for playing on popular VTTs. 

Tis a big undertaking, but I am confident I can get there. 


#### Feature Roadmap

1. Main Page
2. Feature Creator (Everything is a feature)
3. Class Creator (Compose Class Specific Features)
4. Character Creator
5. Character Sheet
6. Dice Rolling
7. Character Export
